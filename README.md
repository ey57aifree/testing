# 🐍 貪食蛇遊戲

一個精美的網頁貪食蛇遊戲，採用純 HTML/CSS/JavaScript 開發。

## ✨ 功能特點

### 核心遊戲功能
- **流暢的蛇移動控制** - 支援方向鍵和 WASD 雙重控制
- **智慧碰撞檢測** - 精確的牆壁與自身碰撞判斷
- **分數系統** - 即時分數顯示與最高分記錄
- **動態難度** - 隨著分數增加，遊戲速度逐步提升

### 使用者體驗
- **精美的 UI 設計** - 現代化介面與流暢動畫效果
- **蛇頭眼睛動畫** - 根據移動方向動態變化
- **食物光暈效果** - 吸引注意的視覺設計
- **響應式設計** - 完美支援桌面與行動裝置

### 操作控制
- **鍵盤控制**
  - `方向鍵` 或 `WASD` - 控制蛇的移動方向
  - `Space` - 暫停/繼續遊戲
  - `R` - 重新開始遊戲

- **觸控控制**（行動裝置）
  - 滑動螢幕控制蛇的移動方向

### 資料持久化
- **本地儲存** - 使用 localStorage 保存最高分記錄
- **持續記錄** - 跨瀏覽 sessions 保留使用者進度

## 🎮 遊戲規則

1. **目標**：控制蛇移動並吃掉食物，獲得最高分數
2. **得分**：每吃掉一個食物獲得 10 分
3. **遊戲結束**：蛇撞到牆壁或自身身體時遊戲結束
4. **速度提升**：每累積 50 分，遊戲速度增加
